# Python Projects

A collection of small Python games, GUI experiments, automation scripts, and data-science notebooks.

## Project Contents

| Project | Description |
|---|---|
| [CaeserCypher](CaeserCypher/) | Command-line Caesar cipher encoder and decoder. |
| [EditingTextFile](EditingTextFile/) | Basic text-file reading and writing examples. |
| [Hangman Game](Hangman_game.py/) | Command-line Hangman game with words and ASCII art. |
| [High Low Game](High_Low_Game/) | Number comparison guessing game with score tracking. |
| [Mail Merge Project](<Mail Merge Project Start/>) | Generates personalized invitation letters from templates and name lists. |
| [Snake Project](SnakeProjectCode/) | Turtle-based Snake game with food, scoring, and collision handling. |
| [Audio Bot](audioBot1/) | Audio recording, speech, waveform analysis, and conversion experiments. |
| [Blackjack Game](blackjack_game/) | Command-line Blackjack game with score and ace handling. |
| [Guess the Number](guessTheNumber/) | Difficulty-based number guessing game. |
| [Pandas Python](pandasPython/) | NumPy, pandas, data-wrangling, and basic neural-network exercises. |
| [PDF Project](pdfProject/) | PDF and HTML template experiments with a starter Python script. |
| [Plot Types](plot_types_jupyter/) | Matplotlib examples for basic, statistical, 3D, array, and unstructured plots. |
| [Python Calculator](pythonCalculator/) | Interactive four-operation command-line calculator. |
| [Coffee Machine](pythonCoffeeMachine/) | Object-oriented coffee-machine components and menu models. |
| [Colorgram](pythonColorgram/) | Turtle dot-art generator using an image-inspired color palette. |
| [Tkinter GUI](pythonGUI_tk/) | Examples of common Tkinter widgets and layout methods. |
| [NATO Alphabets](pythonNATO_Alphabets/) | Starter program that converts input text into a letter list. |
| [Object Detection](pythonObjectDetection/) | OpenCV notebook for multi-object video tracking. |
| [Python OOP](pythonOops/) | OOP and Turtle graphics exercises, including a spirograph. |
| [Question Bank](pythonQuestionbank/) | Object-oriented true/false quiz with score tracking. |
| [CSV Reader](pythonRepharse/) | Tkinter file picker that loads and displays CSV data with pandas. |
| [Turtle Project](pythonTurtleProject/) | Keyboard-controlled Turtle drawing application. |
| [Tkinter Canvas](tkCanva/) | Tomato-themed countdown timer interface prototype. |
